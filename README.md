# Study Time Tracker

A simple Python project that helps students track study time for different subjects.

## Features

- Add study time for subjects
- Generate study report
- Study timer
- Simple command-line interface

## Technologies

- Python

│
├── tracker.py
├── timer.py

├── app.py
└── README.md

## How to Run

python app.py
